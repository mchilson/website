Birmingham, Alabama (June 20th, 2026) **ResizeAll**, the easy-to-use batch image resizer for Windows and Linux, today announced the release of version 0.3.0, bringing usability improvement for web developers and everyday users alike:   


**What’s new in ResizeAll 0.3.0**

- Updated to the latest Dart and Flutter framework version.  
- Optimized Processing Code: Improvements under the hood help ResizeAll run more efficiently during batch operations, reducing overhead while maintaining high-quality resizing results.  
- Updated New About Dialog: A redesigned About window makes it easier to view version details, contact information, and the official project website.  

“ResizeAll is built to be simple, fast, and dependable,” said Mike Chilson, author of ResizeAll. “Version 0.3.0 builds on the existing ease of use for batch resizing images, and improving performance and polishing the overall experience.”  

**Availability**  
ResizeAll 0.3.0 is available now. For more information, visit [resizeall.app](https://resizeall.app).
